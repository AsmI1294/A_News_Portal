The scope of a var variable is functional scope, but The scope of a let and const variables is block scope. var can be updated and re-declared into the scope, let can be updated but cannot be re-declared into the scope, and const cannot be updated or re-declared into the scope.var and let can be declared without initialization, but const cannot be declared without initialization.

this keyword is one of the most used keywords in JavaScript. But when it comes to regular functions and arrow functions, it behaves in entirely different ways. Arrow functions don't have their own “this”, and they don’t redefine the value of “this ”within the function. In regular JavaScript functions, arguments keywords can be used to access the passed arguments when the function is invoked, But, arrow functions do not have their own arguments and it uses the arguments from the outer function.If the arrow function contains one expression, you can omit the curly braces, and then the expression will be implicitly returned.

The map() method in JavaScript returns an array by calling a specific function on each element present in the parent array. The forEach() method calls a function for each element in an array but it does not return anything. The filter() method is used to apply a test to an array. It then returns all the values that pass this test in the form of an array. The find() method returns the value of the first element that passes a test. The find() method executes a function for each array element.

Template literals are literals delimited with backtick (`) characters, allowing for multi-line strings, for string interpolation with embedded expressions, and for special constructs called tagged templates. Template literals are sometimes informally called template strings, because they are used most commonly for string interpolation (to create strings by doing substitution of placeholders). However, a tagged template literal may not result in a string; it can be used with a custom tag function to perform whatever operations you want on the different parts of the template literal.
